Site Description: This is a site mean for players to be able to see the Gods, Items, and Builds for the MOBA Game SMITE. Descriptions for each page will be located below.
Gods/Items pages: Ignore this pages as they have no functionality except to display final results to user

Admin God Table: This page allows Admins to view the available "Gods" table and INSERT new gods if need be also allows for the FILTER of gods based on their class, role, or type

Admin Types Table: This page allows Admins to view the available "Types" in the game and INSERT new types if need be

Admin Classes Table: This page allows Admins to view the available "Classes" in the game and INSERT new classes if need be

Admin Items Table: This page allows Admins to view the available "Items" in the game and INSERT new items if need be

Admin Item Types Table: This page allows Admins to assign existing types to existing items. It will also allow admins to DELETE types from items AND/OR UPDATE types from items. Note: An item must atleast have 1 type.

Admin Roles Table: This page allows Admins to view the available "Roles" in the game and INSERT new roles if need be

Admin Builds Table: This page allows Admins to view the current build for any God and INSERT new items into the build

NOTE: Most functionality resides in the admin pages. Admin pages are not meant to be used to change tables very often. For example it is highly unlikely that new classes and roles will ever be added into the game, but the functionality is there just incase the game undergoes drastic changes. Whereas adding new gods, types, items, etc. is much more likely but only happens a handful of times throughout the year.
